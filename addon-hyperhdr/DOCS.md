[![HyperHDR logo](https://raw.githubusercontent.com/ihrapsa/hassio-addons/master/addon-hyperhdr/logo.png)](https://www.hyperhdr.eu/)

### Installation

Install and start the add-on.
You can open HyperHDR by going to http://HA_IP:8090
