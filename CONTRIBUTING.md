# How to contribute

## Support Requests vs Bug Reporting

Please report any issues with the Home Assistant app. Issues with HyperHDR itself should be posted on the HyperHDR's github page.
If you are not sure if the issue persists across HyperHDR, submit it here but indicate that it requires testing.

## Pull Requests

Contributions to the project are welcome. Please TEST all commits before submitting a pull request.
